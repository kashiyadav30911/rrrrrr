# 🚀 Quick Start Guide

Get your AI Resume Builder running in 5 minutes!

## Step 1: Install Dependencies

```bash
npm install
```

## Step 2: Get Your OpenRouter API Key

1. Go to https://openrouter.ai/
2. Sign up / log in and create an API key
3. Copy the key

## Step 3: Create Environment File

Create a file named `.env.local` in the root directory:

```env
OPENROUTER_API_KEY=paste_your_key_here
```

## Step 4: Run the App

```bash
npm run dev
```

## Step 5: Open in Browser

Visit: `http://localhost:3000`

## 🎉 You're Ready!

1. Click "Create Resume" on the homepage
2. Fill in your information
3. Click "AI Enhance" to generate professional content
4. Download your resume as PDF

## 💡 Tips

- **Save Often:** Click the "Save" button to store your progress
- **Use AI Wisely:** Provide detailed info for better AI suggestions
- **Review Output:** Always review and customize AI-generated content
- **Dark Mode:** Toggle dark mode with the moon/sun icon

## 🐛 Common Issues

### "API key not configured"
- Make sure `.env.local` file exists
- Check the API key is correct
- Restart the dev server

### "Failed to generate AI content"
- Verify your API key is active
- Check you have API quota remaining
- Try again in a few seconds

### PDF not downloading
- Check browser allows downloads
- Try a different browser
- Look for download in your Downloads folder
